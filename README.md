# WebPageCenote

Website for Cenote. This will create the website where a user can download their Cenote application and get information about the product. 

## Potential routes and features


cenote.study/
Long read describing the application.

cenote.study/download
This is where you download the application. 

dashboard.cenote.study/profile
Login route for users 

dashboard.cenote.study/sessions/
Shows the sessions user has created

dashboard.cenote.study/sessions/{id}
shows specific sessions



## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
